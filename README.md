# Tetriks

Tetriks is a Tetris remake made in C using OpenGL and love for the greatest game ever made.

## Installation
### On Linux:
Download files, make files, run it, and have fun!

```bash
make tetriks
./tetriks
```
### On Windows:
Download the windows64 folder and run exe!
